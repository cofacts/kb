Talk：我城對話 : LLM 如何協助消滅共識幻覺 Talk to the City : How LLM can Help Discriminate Consensus Illusion
共筆：https://g0v.hackmd.io/@summit2024/notes/%2FMCsg1l10SuaOdoH3Onyn0g
逐字稿案例 https://talktothecity.org/report/ai-assembly-2023-workshops_1
影片案例 https://talktothecity.org/report/heal-michigan

- Input：審議逐字稿或影片
- Output：LLM 議題整理
- Prompt 在下面
- Insight
  - 如果想要 LLM 特別加強什麼或排除什麼，那就要寫在 prompt 裡面
  - 寫在 prompt 裡面會很明顯，所以把 prompt show 出來讓大家來看，有助於讓讀者知道說可能有什麼 bias
- For Cofacts
  - summarization, claim extraction 的 prompt 在做 [LLM analysis](https://g0v.hackmd.io/mU8qi721RZeAQ9PDfj7XRA#UX-on-website) 時很有用
  - show 出 prompt 的部分，跟 Cofacts 未來把查核協作者[對 LLM 的溝通](https://g0v.hackmd.io/mU8qi721RZeAQ9PDfj7XRA#UX-on-website) 公開有異曲同工之妙
