## GA4 <> UA

> 承 3/1 時的比較 https://g0v.hackmd.io/zVyuEX5RS5uONLnucsxiYA#GA4-ltgt-UA-%E5%B7%AE%E7%95%B0

### LIFF

Both UA (`ViewArticle` events) and GA4 (`view_item` events) have exactly the same trigger.
![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_89b42d5418a2834aece089487f350ecb.png)


以下以 article `2q2xw134jyblq` 為例。

UA:

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_b841d09fef96669b7197c74e2d4b0a26.png)
https://lookerstudio.google.com/u/0/reporting/18J8jZYumsoaCPBk9bdRd97GKvi_W5v-r/page/p_44cex908mc?params=%7B%22df320%22:%22include%25EE%2580%25800%25EE%2580%2580IN%25EE%2580%25802q2xw134jyblq%22%7D

SQL:

```sql
SELECT event_date, count(distinct user_id) as user_count, count(*) as event_count
FROM `<table>.events_*`, UNNEST(items) as items
WHERE stream_id = '<LIFF stream ID>' AND event_name = 'view_item' AND items.item_id='2q2xw134jyblq'
GROUP BY event_date
ORDER BY event_date ASC
```

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_0392cc574346e05d2fb9ab98d4adf6ef.png)

Result: https://docs.google.com/spreadsheets/d/18jX_N5shuJHibWHJlH9YXfB9Mo-th8d7SdZVFbIQpBI/edit#gid=741917313
- GA4 gets 1.45% less event
- GA4 gets 0.5% more unique users (?)
  - 可能是因為 GA4 是 unique user count 每日相加，UA 是一整個區段的 unique user count
