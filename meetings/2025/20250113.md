### [Op] Automated spam removal

> Design doc: https://g0v.hackmd.io/@cofacts/rd/%2Fum7DyY_ESbu2LL78kLw3pg

改成 openai 版本 
- create 了一個新的 openai project：`SpamDetection`
- [version 1](https://langfuse.cofacts.tw/project/cm3n1h0xu000mfdgamwhs63t2/traces) 每則訊息都是一個 trace，比較不好 review
- 覺得 gemeni 的版本，會把同時 generate 的 chat group 成同一個 trace，比較好看，所以有了 [version2](https://langfuse.cofacts.tw/project/cm3n1h0xu000mfdgamwhs63t2/traces/e1976495-5da2-4922-a6e8-011c0a87a429)

測試結果
- version 1 沒有加 few shot example，目前的 test case 有兩個 false positive
- version 2 加了 few shot example，目前的 test case 皆能通過
