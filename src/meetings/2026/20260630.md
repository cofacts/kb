# 【事件紀錄】LINE OA 統計資料中斷（2026-05-05 ~ 2026-06-30）

**影響範圍**
Cofacts 統計試算表（Google Apps Script）無法抓取 LINE OA 粉絲數據，試算表 Followers 頁的資料在 2026-05-04 後即停止更新。

**根本原因**
2026-05-05 啟用 Cloudflare Super Bot Fight Mode（SBFM）時，Google Apps Script 被誤判為惡意 bot：

- Google Apps Script 發出的 HTTP 請求含特殊 User-Agent（`Google-Apps-Script; beanserver`）
- Cloudflare 的 "Bot Fight Mode for Definite Bots" 規則對其發出 Managed Challenge（JS 挑戰）
- Google Apps Script 是純 HTTP 請求，無法執行 JS，挑戰必敗 → 收到 403 Forbidden
- Cloudflare 並未將 Google Apps Script 列為 Verified Bot，故無法透過一般白名單機制處理

**處置**
在 WAF Custom Rules 的「Bot Fight Mode false positives」規則中，將 `line-bot.cofacts.tw/graphql` 加入 SBFM 跳過清單。2026-06-30 修復後確認資料已恢復正常回填。

**補充**
事發期間約 **56 天**的 LINE OA 統計資料（5/5 ~ 6/29）未被記錄進試算表，這段時間的 followers 趨勢需從 LINE OA 後台或 Cloudflare analytics 另行補充。
