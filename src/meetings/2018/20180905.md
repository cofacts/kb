## URL Preview

### Done

PR 求 review:
- 串接 url-resolver 與 rumors-api https://github.com/cofacts/rumors-api/pull/103


更新了 url-resolver 
- 自動展開短網址（unshorten）
- 若 URL 長得像 youtube，就直接用 youtube API 抓 title & description & 圖！
- 遇到 `confirm()` / `alert()` 會自動按掉！（不然會整個卡住）

寫好了一個 script

- 掃所有 article & reply，寫入 `urls` index 與 `hyperlinks` field
- 如果 `urls` 有 cache 過，就用 cache 裡的來寫 `hyperlinks` field
- 但 url-resolver 需要更好的 error handling
- script 跑完約需 1 day
- Migration 時，會把跑好的 `urls` index 直接塞進 production DB，然後進 docker container 再執行一次 script
