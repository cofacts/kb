#### From LINE bot analytics

- Document GA tags in README (https://github.com/cofacts/rumors-line-bot/pull/69/files)

- Hitrate = `UserInput/ArticleSearch/ArticleFound` / (`UserInput/ArticleSearch/ArticleFound` + `UserInput/ArticleSearch/ArticleNotFound`)
- article submission rate = `Article/Create/Yes` / (`Article/Create/Yes` + `Article/Create/No`)

- has article funnel (bar chart) = `UserInput/MessageType/text` -> `UserInput/ArticleSearch/ArticleFound` -> `Article/Selected/*` -> `Reply/Selected/*` -> `UserInput/Feedback-Vote/*`

- Searched article `Article/Search/*` label rank
- selected article `Article/Selected/*` rank 
- Selected reply `Reply/Selected/*` rank
