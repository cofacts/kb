## Deploying new schema

SOP:

1. Take down LINE & website
```
# Local
$ heroku maintenance:on -r heroku

# Production
$ docker-compose stop nginx
```

2. Backup rumors DB

Follow https://github.com/cofacts/rumors-db/issues/7

then pack the snapshot of the new DB

```bash
DELETE /_snapshot/my_backup/snapshot_1

PUT /_snapshot/my_backup/snapshot_2?wait_for_completion=true
{
  "indices": "*_v1_0_0"
}
```

Delete `nodes`, then restore from snapshot

Pack the new `nodes` it up then upload to production server


3. Setup up new rumors DB

```
$ Production
docker-compose up db
```

4. Spin up new site, API server

5. Spin up new LINE bot

6. Take system back online
